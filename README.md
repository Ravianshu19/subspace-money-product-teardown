# Subspace.money Product Teardown

## Objective

This product teardown was created for the Product Intern Assignment.

The goal was to evaluate Subspace.money across:
- UX
- GTM & ICPs
- Features & Services
- Trust & Security
- Growth & Collaborations

---

## Research Methodology

### Product Exploration

Explored the application as a first-time user and analyzed:

- Home Screen
- Explore Page
- Wallet
- Chat
- Account & KYC

### Frameworks Used

- User Journey Mapping
- SWOT Analysis
- JTBD (Jobs To Be Done)
- Competitive Benchmarking

---

## Key Findings

### 1. Empty State Problem in Chat

**Observed**

The Chat section displays "No conversations found" with no onboarding guidance.

**Problem**

Users don't understand how to begin interactions or why the feature exists.

**Ship Instead**

- Suggested groups
- Create Group CTA
- Invite Friends CTA
- Onboarding flow

---

### 2. Wallet Activation Friction

**Observed**

New users see ₹0 balance, ₹0 transactions, and large empty sections.

**Problem**

The page fails to communicate wallet value.

**Ship Instead**

- Cashback examples
- First transaction reward
- Wallet benefits section

---

### 3. Product Positioning Confusion

**Observed**

Homepage promotes gift cards, credit cards, payments, subscriptions, and rentals simultaneously.

**Problem**

Users struggle to understand the primary value proposition.

**Ship Instead**

Create a clear hierarchy:

1. Subscription Sharing
2. Gift Cards
3. Rentals

---

### 4. KYC Trust Opportunity

**Observed**

Large KYC banner lacks clear explanation.

**Problem**

Users perceive KYC as friction.

**Ship Instead**

Show benefits:
- Verified Seller Badge
- Faster Withdrawals
- Fraud Protection

---

### 5. Discovery Experience

**Observed**

Explore page contains many categories but limited personalization.

**Problem**

Choice overload reduces conversion.

**Ship Instead**

- Trending Products
- Recommended For You
- Popular This Week

---

## Priority Roadmap

### Phase 1

- Wallet Activation
- Chat Onboarding

### Phase 2

- KYC Trust Layer
- Discovery Personalization

### Phase 3

- Recommendation Engine
- Social Sharing Features
